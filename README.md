## GoToMountain

## Overview
젊은 등산객들이 많아지면서 좀 더 쉽게 등산에 참여할 수 있도록 정상 까지의 소요 시간과 주위 캠핑장 위치 등을 알려주는 어플리케이션입니다.

## 참조
- 주소에 따른 위도와 경도를 구하기 위해 Geocoder API를 활용하였습니다. 
- 산의 구간 별 예상 소요시간을 표시해주기 위해 KAKAO MAP을 활용하였습니다. 

## 느낌점
 데이터 수집에 있어서 산도 많고 정확하지 않은 정보도 있어서 조금 어려움이 있었던 것 같습니다. 그리고 저희 자체의 소요시간을 구하는 기능을 만들어보고 싶었지만 너무 어려워 결국 KAKAO API를 사용했다는 아쉬움이 컸었던 것 같습니다. 혹시나 나중에 자바 언어에 대한 역량이 엄청 성장한다면 직접 만들어보고 싶은 생각이 듭니다.
