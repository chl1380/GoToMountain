## GoToMountain

## Overview
젊은 등산객들이 많아지면서 좀 더 쉽게 등산에 참여할 수 있도록 정상까지의 경로와 주위 캠핑장등을 알려주는 어플리케이션입니다.

## 참조
- 주소에 따른 위도와 경도를 구하기 위해 Geocoder API를 활용하였습니다. 
- 산의 구간 별 예상 소요시간을 표시해주기 위해 KAKAO MAP을 활용하였습니다.
